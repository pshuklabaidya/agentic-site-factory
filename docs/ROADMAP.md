# Roadmap

## Completed Baseline

- Streamlit dashboard.
- TXT and PDF ingestion.
- TF-IDF retrieval.
- Agent-style planning.
- Deterministic section generation.
- Optional OpenAI section generation.
- Custom style-aware static website rendering.
- Demo commerce section.
- Quality checks.
- Artifact manifest.
- Evidence map.
- Full artifact ZIP download.
- Command-line demo generation.
- JSON specification-driven generation.
- Pytest and Ruff checks.
- GitHub Actions CI.

## Near-Term Enhancements

- Add richer book catalog extraction.
- Add user-editable section copy before export.
- Add image upload support for gallery and book covers.
- Add optional deployment export for Netlify or GitHub Pages.
- Add stronger hallucination checks against retrieved evidence.
- Add multi-agent trace visualization in the dashboard.
- Add side-by-side comparison across themes.

## Advanced Enhancements

- Add vector database storage for larger document collections.
- Add structured manuscript metadata extraction.
- Add generated sitemap.xml and robots.txt.
- Add generated product schema markup.
- Add Stripe, Shopify, or Gumroad integration stubs.
- Add browser-based preview tests.
- Add Docker support.


- Custom aesthetics are inferred from the website specification, optional style guidance, retrieved passages, and uploaded source material.


- The generated artifact bundle includes theme_spec.json with the inferred visual style rationale and CSS values.
